# Other useful Unix commands

-   `clear` - clear the terminal output
-   `history` - see all the commands typed so far
-   `dos2unix` - clean files edited in MS Windows notepad for Unix (see
    also unix2dos)
-   `echo` - print a message (useful in scripts)
-   `sort` - Sort the lines in stdin, and print the result to stdout.
-   `uniq` - Read from stdin and print unique (that are different from
    the adjacent line) to stdout.
-   `cat` - Read lines from stdin (and more files), and concatenate them
    to stdout.
